# jueguitos 🎮

Colección de juegos educativos web, autocontenidos y sin backend. Cada juego vive en su
propia carpeta y se ejecuta simplemente abriendo su `index.html` (funciona offline).

## Juegos

| Juego | Carpeta | Descripción |
|-------|---------|-------------|
| 🏝️ La Isla de las Palabras | [`isla-de-las-palabras/`](./isla-de-las-palabras/) | Juego de escritura manual y ortografía para niños de 5 a 7 años. Dictado por voz, trazo con el dedo/lápiz, retroalimentación positiva y una isla que se construye con cada acierto. |

## Filosofía técnica

- HTML5 + CSS3 + JavaScript (ES6+), sin frameworks pesados.
- Sin backend, sin servidor, sin base de datos remota.
- Todo el progreso se guarda localmente en el navegador.
- Funciona 100% offline una vez cargado.
- Arquitectura pensada para empaquetar a móvil (WebView / app híbrida) en el futuro.
